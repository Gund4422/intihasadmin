# Intiha's Admin
Most admin scripts (like Infinite Yield) are built on legacy architectures that prioritize "more code" over "fast code." On high-end rigs, you might not notice the bloat—but on an i3-5005U with 8GB RAM, it’s the difference between a smooth game and a total crash.The $O(1)$ AdvantageIA utilizes a Direct-Mapped Hash Table for command execution.Infinite Yield ($O(n)$): Uses a linear search. To run a command, the CPU must iterate through a massive array of strings, comparing your input to every single entry. If the command is at the bottom of the list, your i3 is performing hundreds of redundant checks while the game engine is already starving for cycles.Intiha’s Admin ($O(1)$): Uses constant-time lookups. Your command is a "Key" that points directly to a "Value" (the function) in memory. Whether there are 10 commands or 1,000, the lookup time is identical.
